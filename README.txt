
# Alternative Engineered Fuel – Deploy Package

Contents:
- index.html (main integrated research portal)
- advanced_energy_site.html
- fuel_comparison_dashboard.html
- scenario_simulator_2050.html
- animated_sankey_energy_flow.html

Deployment:

1. Create a GitHub repository
2. Upload all files
3. Enable GitHub Pages:
   Settings → Pages → Deploy from branch → main → /(root)
4. Wait 1–3 minutes
5. Open:
   https://YOUR_USERNAME.github.io/REPOSITORY_NAME/
